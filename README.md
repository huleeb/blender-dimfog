# Blender Dim Fog theme

Compatible with Blender 2.83+

![Dim Fog](https://raw.githubusercontent.com/huleeb/blender-dimfog/refs/heads/assets/dim_fog_default_blender.png)

Installation
------------

Theme is available through [Blender official theme website](https://extensions.blender.org/themes/).

- [Dim Fog](https://extensions.blender.org/themes/dim-fog/)
